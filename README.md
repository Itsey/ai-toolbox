# ai-toolbox

A collection of AI skills and tools for use with **Claude** (Anthropic) and **GitHub Copilot**.

## Structure

```
skills/
├── claude/     # Skills for Claude (Anthropic tool use)
├── copilot/    # Skills for GitHub Copilot (Copilot Extensions)
└── shared/     # Skills that work across both platforms
```

See [`skills/README.md`](./skills/README.md) for full documentation on how the skills are organized and how to add new ones.
